FlexGP: Flexible Machine Learning with Genetic Programming

FlexGP is a Genetic Programming framework aimed at data modeling and knowledge discovery. Current release provides functionality both for performing Symbolic Regression on numerical datasets and for testing the retrieved models. In this website we provide a quick tutorial on how to get started with FlexGP.

Current release

The functionality of the current release is limited to Symbolic Regression tasks in a single desktop (capability to attack binary classification problems and extensions to support cloud-based executions will be added soon).

Tutorial

We provide a tutorial for the different core learners developed within the frame of the FlexGP project. Additionally, we provide a guide to get started with FlexGP and deploy your runs in cloud environments.

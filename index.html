FlexGP: Flexible Machine Learning with Genetic Programming

FlexGP is a long-term project aimed at data modeling and knowledge discovery with evolutionary computation. Within the scope of FlexGP, we have developed a number of evolutionary learners and two cloud platforms for large-scale machine learning.

FlexGP learners

We provide an overview and tutorial of the different core learners developed within the frame of the FlexGP project.

Cloud-based platforms

FlexGP: a cloud platform for large scale regression problems

FCUBE: a cloud platform for large scale classification problems
